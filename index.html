The new way to consume media! This is a flask based web app designed to allow users to search for a topic of interest and generate an instagram story from the given query. Users can select from a wide variety of articles and download an the instagram story directly to their phone and/or mobile device. As part of the backend team, I am using the PIL library to perform the image processing and generation.

A fun project to learn more about chat bots. This bot allowed viewers in a stream to send compliments to one another. Additionally, jokes could be exchanged between viewers and the host to enhance the streaming experience. Features such as song requests are currently in development.
